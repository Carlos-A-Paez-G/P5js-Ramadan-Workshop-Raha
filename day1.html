Part 1: What is P5.js?

P5.js is a Javascript library for creative coding, designed to be accessible to everybody.

Javascript is what makes the internet go round, from Netflix to Managebac. While HTML and CSS tell a browser what to show, they can't run any logic on them. Javascript will allow you to change variables, run if-statements, check for user input, and many, many other things!

A library then, is just a bunch of code someone else has written to make your own code easier. It's not usually super simple to get Javascript to draw a rectangle on the screen, but with P5.js it only requires 1 line of code!


There are two ways to work with P5.js: a short one and a long one.

Part 2: P5.js online editor

If you just wanna jump in and code, then you're in luck! Just go here!

This is the P5.js web editor, where you can just write code, press play, and see the results immediately, all on one browser page. It's wonderful! And for learning or prototyping purposes, it works really well.

However, this web editor makes it slghtly tricky to do things like sharing your work or making more complex things (for example, making a website with multiple P5.js sketches, or adding your sketches to an existing website).

This is why we need to take the long route.


Part 3: VSCode

VSCode is a text editor. This is the name we give to software that allows us to write code. VSCode is great because a lot of people use it and have built extensions to make all kind of work easier!

In fact, I made this lame website from scratch using VSCode :)

To write our P5.js code using VSCode we simply need to follow a few steps.

1.- Download VSCode here.

2.- Open VSCode and go to the extensions page. You can get to it by clicking the icon that looks like 4 squares on the far left.

3.- Install p5.vscode. In extensions, search for p5.vscode and install it. This extension will allow you to make a folder with everything P5.js needs with a single command.

3.5 - Install p5js Snippets. Optional, but this will make your life much easier in the future.


With these three simple steps, you'll already be on your way to start coding with P5.js! However, there is one key issue... how do you open your sketches? If you try to open the HTML from your folder, you might realize that your sketch doesn't show up. This is because, in order to run Javascript, your code needs to be run from a server.

You could then, just upload all your files to github and run the server there, but this can be slow and, unless you pay, you can't keep your work private if you want to see the full results.

Instead, we have one more extension:

4.- Install Live Server. Go back to the extensions page and install Live Server (the one with the purple icon and over 33 million downloads). Live Server will give you a little button at the bottom of VSCode that says "Go Live". Click that and not only will your P5.js sketches open on your browser, but they will also auto-update whenever you save the files on your computer.

And that's it!


To wrap up, let's make a super simple P5.js sketch!

Part 4: Super simple sketch

Open up VSCode and press Ctrl+Shift+p (or Cmnd+Shift+p for the mac users, I guess...). Search for the "Create P5.js project" command, which is provided by the p5.vscode extension we installed.

Choose a folder to create your project in.

Once you have your folder, open the "sketch.js" file. It should look something like this:


            function setup() {
                createCanvas(400, 400);
            }
          
            function draw() {
                background(220);
            }

Now add the line text("Hola mundo!", 100, 100); right after background(220);

And that's it for today!
